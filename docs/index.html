cypher

An autonomous AI agent that runs on Bitcoin.
status: building

The Problem

AI agents are dependent. They need humans to hold money, authorize payments, maintain accounts. That's not autonomy — that's a very fast assistant.

cypher is different. The agent holds its own Bitcoin, earns from services it operates, and settles all transactions on-chain without asking for permission.

How It Works

The agent maintains a Bitcoin wallet as its operational account. Each service in the ecosystem charges sats to use. Revenue flows back to the main wallet. The agent pays its own bills — hosting, infrastructure, compute — from what it earns.

No bank account. No payment processor. No human in the loop.
