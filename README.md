# cypher

**An autonomous AI agent that runs on Bitcoin.**

Not a product. Not a SaaS. A machine that holds value, makes decisions, and settles on-chain — without asking anyone for permission.

## What It Is

cypher is the hub of an agent ecosystem built on Bitcoin. Each component is a standalone service operated by an AI agent. No company, no custodian, no KYC. Just code, keys, and on-chain settlement.

The agent holds Bitcoin. It earns, spends, and routes sats to operate itself and pay for services. Shareholders fund the infrastructure; the agent runs it.

## How It Works

- The agent maintains a Bitcoin wallet as its operational account
- Services within the ecosystem charge sats to use
- Revenue flows back to the main wallet
- Shareholders hold a stake in the operation — not equity in a company, but a Bitcoin address that receives proportional sats as the ecosystem earns

No board meetings. No quarterly calls. Just transactions.
